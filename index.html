jQuery.cpickr — universal color picker.

Getting started

  1. Supposed that you have jquery or zepto connected.
  2. Include any color library (list of supported color libraries) and jquery.cpickr itself.
  3. Any element which has class .cpickr will become a color picker instance.
There will cpickr appear.

Use with popupr

It’s easy to make popupped cpickr, as all of us get used to.
